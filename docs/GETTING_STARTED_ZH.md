## 🚀 第一章：唤醒你的 Agent

虽然安装程序已经帮你配好了环境，但你还需要执行一次“唤醒仪式”来给它注入灵魂：

1. **执行命令**：
   在你的终端输入：`npx openclaw onboard`
2. **配置向导**：
   你会看到一个很酷的命令行界面，按照提示完成以下配置：
   - **API Key**：填入你的大模型密钥（推荐 Anthropic Claude 或 DeepSeek）。
   - **Telegram Token**：去 [@BotFather](https://t.me/BotFather) 申请一个机器人，把那一串代码贴进来。
3. **开始聊天**：
   配置完成后，去 Telegram 找到你的机器人，发一句：“你好”，它就会正式为你效劳了！

## 🛠️ 第二章：常用维护指令 (备忘录)

你不需要记住复杂的代码，只需在终端敲这几行：

- **查看状态**：`npx openclaw status` (看看 Agent 是否在线)
- **重启 Agent**：`npx openclaw restart` (配置修改后生效)
- **检查更新**：`npx openclaw update` (获取最新功能)
- **查看医生报告**：`npx openclaw doctor` (如果它“生病”不回消息，用这个自检)

## 💡 第三章：如何让它更好用？(进阶技巧)

1. **给它起个名字**：
   修改 `~/.openclaw/workspace/IDENTITY.md`。你可以把它叫作“贾维斯”、“星期五”或者任何你喜欢的名字。
2. **定义它的灵魂**：
   打开 `~/.openclaw/workspace/SOUL.md`。在里面写上：“你是一个说话幽默、办事利索的私人助理。”它以后说话的语气就会随之改变。
3. **给它分配任务**：
   在 Telegram 里对它说：“以后每天早上 8 点，帮我搜一下关于 [XXX] 的最新热点并整理成报告发给我。”它会利用内置的 **Proactive Monitoring** 技能自动执行。

## 🛡️ 第四章：安全与隐私

- **API 安全**：ClawEase 内置了 **Security Auditor**。如果它发现你安装的某个新技能试图偷看你的配置文件，它会立刻拦截。
- **数据存储**：记住，所有的聊天记录和 Key 都只存放在你自己的电脑（`~/.openclaw` 文件夹下）。Tony V 不会把你的秘密上传到任何第三方服务器。
